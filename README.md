# OLA Ride Booking Data Analysis

## Project Overview
___________________________________________________________________________________________________
<img width="1536" height="1024" alt="Ola Final Banner" src="https://github.com/user-attachments/assets/080fee5c-c992-4e5b-a03d-1fe8fc657ae2" />

This project analyzes OLA ride booking data using SQL, Excel, and Power BI to generate actionable business insights. 
The objective is to understand booking trends, revenue performance, cancellation patterns, customer behavior, vehicle utilization, and service quality.

The project demonstrates the complete Data Analytics workflow including data cleaning, SQL analysis, KPI creation using DAX, and interactive dashboard development.

---

## Problem Statement

Ride-sharing companies generate large volumes of operational data every day. The objective of this project is to transform raw booking data into meaningful insights that help improve operational efficiency, customer satisfaction, revenue generation, and decision-making.

The dataset contains ride booking information including:

* Booking Value
* Ride Distance
* Payment Method
* Driver Ratings
* Customer Ratings
* Cancellation Reasons

Dataset Size:

* 20,407 Records
* 19 Columns

## Key Insights

* Total Bookings: 20,407
* Total Revenue: 7M+
* Cancellation Rate: 28.10%
* Cash and UPI generated the highest revenue.
* Prime Sedan contributed the highest booking value.
* Average ratings remained close to 4.0 indicating stable service quality.
